# Customer Segmentation & Churn Pattern Analytics in European Banking

## Project Overview
This project is an interactive Streamlit dashboard developed to analyze customer churn patterns in a European bank. The dashboard helps identify customer segments with high churn risk, compare churn across different regions, and provide business insights for improving customer retention strategies.

## Objectives
- Measure overall customer churn rate.
- Analyze churn across customer segments.
- Compare churn behavior across European regions.
- Identify high-value customers at risk of churn.
- Evaluate customer engagement and tenure patterns.
- Support business decision-making using interactive visualizations.

## Dataset
The project uses a European banking customer dataset containing information such as:
- Geography
- Gender
- Age
- Credit Score
- Balance
- Tenure
- Number of Products
- Active Member Status
- Estimated Salary
- Customer Churn (Exited)

## Installation

1. Clone the repository
```bash
git clone https://github.com/neelakantam2/Customer-Churn-Analytics.git
```

2. Install dependencies
```bash
pip install -r requirements.txt
```

3. Run the application
```bash
streamlit run app.py
```
